# Y-Combinator-Clone

## Overview
This project is a high-fidelity clone of the Y Combinator landing page. It showcases modern web design practices, focusing on rich aesthetics, smooth transitions, and responsive layouts to closely recreate the premium feel of the original website. The site features various dynamic sections including a company showcase with scroll-triggered animations, interactive founders grids, and embedded media elements.

## Tech Stack
- **Frontend Framework**: React.js (Bootstrapped with Vite)
- **Styling**: Vanilla CSS (Custom properties, Flexbox, CSS Grid, Media Queries)
- **Language**: JavaScript (ES6+)

## Folder Structure
```text
src/
├── assets/             # Static assets like images, SVGs, and logos
├── components/         # Reusable React components organized by page section
│   ├── CompanyShowcase/
│   ├── FoundersComments/
│   ├── FoundersGrid/
│   ├── Hero/
│   ├── History/
│   ├── Knowledge section/
│   ├── Layout/         # Core layout components like Navbar and Footer
│   └── PartnersGrid/
├── styles/             # Component-specific Vanilla CSS files
├── App.jsx             # Main application component assembling all sections
└── main.jsx            # React application entry point
```
